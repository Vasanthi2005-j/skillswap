# 🚀 SkillSwap – Student Skill Exchange Platform

SkillSwap is a modern React-based web application that enables students to exchange skills, connect with peers, track learning progress, earn Skill Coins, and get AI-powered skill recommendations in a collaborative environment.

## 🎯 Objective

SkillSwap solves the problem where students:
- Have skills but no platform to share them
- Struggle to find learning partners
- Lack motivation tracking systems
- Need personalized learning guidance

It builds a **peer-to-peer skill exchange ecosystem**.

---

## ✨ Features

### 🏠 Home Page
- Modern landing page
- Hero section with CTA
- Feature highlights

### 📊 Dashboard
- Student statistics
- Skill matching insights
- Leaderboard system
- Activity tracking

### 👤 Profile System
- Add skills to teach
- Add skills to learn
- Skill Coin system:
  - +20 for teaching
  - -10 for learning request

### 🤝 Skill Exchange
- Browse students
- View skills
- Connect with peers

### 💬 Chat System
- Chat UI simulation
- AI mentor responses
- Learning guidance

### 🤖 AI Recommender
- Suggests skills based on interest:
  - Frontend → React, JS, CSS
  - Backend → Node.js, Express, MongoDB
  - AI → Python, ML
  - UI/UX → Figma, Design Systems

### 🏆 Leaderboard
- Ranked by Skill Coins
- Gamified learning system
